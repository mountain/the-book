异星杂谭
========

幼时夏夜，在屋顶乘凉，仰望壮丽的星空，时而会有一些遐想：在某个闪亮的星星附近，是否也有另一个“我”，同样在仰望星空，同样在发问？
后来读到 Ted Chiang 的《你一生的故事》，深为外星异质文化带来的观念冲击而着迷。为什么呢？因为人们可以通过不同来重新审视自己。
